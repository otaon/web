基本構文

演算子

Goでは下記の演算子が使用できる。

種類   演算子
算術   + - * / % & | ^ ^& << >>
比較   == != < <= > >=
論理   || &&
単行   + - ! ^ * & <-


優先度   演算子
高   * / % << >> & &^
  + - | ^
  == != < <= > >=
  &&
低   ||

演算子の意味と対象の型

演算子   意味   対象の型
+(単項)   意味なし(0 + xと等価)   整数、浮動小数点数、複素数
-(単項)   符号反転   整数、浮動小数点数、複素数
^(単項)   ビットの補数   整数、浮動小数点数
+(2項)   和   整数、浮動小数点数、複素数、文字列(文字列結合)
-(2項)   差   整数、浮動小数点数、複素数
*(2項)   積   整数、浮動小数点数、複素数
/(2項)   商(商は0の方向へ切り捨て/切り上げされる)   整数、浮動小数点数、複素数
%(2項)   剰余   整数
&(2項)   論理積(ビット演算)   整数
|(2項)   論理和(ビット演算)   整数
^(2項)   排他的論理和(ビット演算)   整数
&^(2項)   ビットクリア(a &^ b == a & (^b))   整数
<<(2項)   左シフト(ビット演算)   整数
>>(2項)   右シフト(ビット演算)   整数
== 等しい 比較可能な型
!= 等しくない 比較可能な型
< 未満 順序が有る型
<= 以下 順序が有る型
> 超 順序が有る型
>= 以上 順序が有る型
&& and(論理値) 論理値
|| or(論理値) 論理値
! not(論理値) 論理値


比較可能な型
論理値、整数値、浮動小数点数、複素数、文字列
順序が有る型
整数値、浮動小数点数、文字列

NOTE a = a [演算子] bという形の2項演算子はa [演算子]= bという短縮形を持つ。
